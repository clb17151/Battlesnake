Minimax Algorithm in Python

  • by Administrator
  • Computer Science
  • February 5, 2020February 5, 2020

I will implement a minimax algorithm with alpha-beta pruning and cutoff in this tutorial. This algorithm will be implemented in Python and I am creating an AI that is playing Tic-Tac-Toe and similar games against you or another human.

The minimax algorithm is used to solve adversial search problems in which goals of agents are in conflict, this is the case for most games. In a game with two players, the algorithm assumes that one player wants to maximize (MAX) the score and that the other player wants to minimize (MIN) the score. The algorithm can be extended to include more than two players.

The minimax algorithm evalutes all possible plays from the current game state down to the end of each game and then backes up the values to the current state. The algorithm assumes that both players plays optimal, the MAX player will chose the play with the highest score and the MIN player will chose the play with the lowest score. In a Tic-Tac-Toe game with 3×3 squares, it is 362 880 (9!) different states to examine when the game begins.

Alpha-Beta pruning and cutoff is used to speed up the search time for minimax. Alpha-Beta pruning means that we can ignore searching through nodes that never will be chosen given the information we already have, both players is assumed to play optimal. Cutoff means that we can use a fast heuristic evaluation algorithm when we have reached a certain depth in the minimax search. Cutoff is used in the early phases of a game when the number of possible plays is very high.

Tic-Tac-Toe and similar games

This code can be used for Tic-Tac-Toe, Connect Four and similar games. The AI-player uses a minimax algorithm to take decisions on each turn, a human player will get a recommendation before each move. An evaluation function (get_best_move) is used at cutoff, the evaluation function uses winning positions and heuristics to take decisions. A Connect Four game would be impossible to play without the evaluation function.

  1. # Import libraries
  2. import sys
  3. import random
  5. # This class represent a tic tac to game
  6. class TicTacToeGame:
  8.     # Create a new game
  9.     def __init__(self, rows:int, columns:int, goal:int, max_depth:int=4):
  10.         
  11.         # Create the game state
  12.         self.state = []
  13.         self.tiles = {}
  14.         self.inverted_tiles = {}
  15.         tile = 0
  16.         for y in range(rows):
  17.             row = []
  18.             for x in range(columns):
  19.                 row += '.'
  20.                 tile += 1
  21.                 self.tiles[tile] = (y, x)
  22.                 self.inverted_tiles[(y, x)] = tile
  23.             self.state.append(row)
  25.         # Set the number of noughts and crosses in a row that is needed to win the game
  26.         self.goal = goal
  28.         # Create vectors
  29.         self.vectors = [(1,0), (0,1), (1,1), (-1,1)]
  31.         # Set lengths
  32.         self.rows = rows
  33.         self.columns = columns
  34.         self.max_row_index = rows - 1
  35.         self.max_columns_index = columns - 1
  36.         self.max_depth = max_depth
  38.         # Heuristics for cutoff
  39.         self.winning_positions = []
  40.         self.get_winning_positions()
  42.         # Set the starting player at random
  43.         #self.player = 'O'
  44.         self.player = random.choice(['X', 'O'])
  46.     # Get winning positions
  47.     def get_winning_positions(self):
  49.         # Loop the board
  50.         for y in range(self.rows):
  51.             for x in range(self.columns):
  52.                 
  53.                 # Loop vectors
  54.                 for vector in self.vectors:
  55.                     
  56.                     # Get the start position
  57.                     sy, sx = (y, x)
  59.                     # Get vector deltas
  60.                     dy, dx = vector
  62.                     # Create a counter
  63.                     counter = 0
  65.                     # Loop until we are outside the board
  66.                     positions = []
  67.                     while True:
  69.                         # Add the position
  70.                         positions.append(self.inverted_tiles.get((sy, sx)))
  72.                         # Check if we have a winning position
  73.                         if (len(positions) == self.goal):
  75.                             # Add winning positions
  76.                             self.winning_positions.append(positions)
  78.                             # Break out from the loop
  79.                             break
  81.                         # Update the position
  82.                         sy += dy
  83.                         sx += dx
  84.                 
  85.                         # Check if the loop should terminate
  86.                         if(sy < 0 or abs(sy) > self.max_row_index or sx < 0 or abs(sx) > self.max_columns_index):
  87.                             break
  89.     # Play the game
  90.     def play(self):
  92.         # Variables
  93.         result = None
  95.         # Create an infinite loop
  96.         print('Starting board')
  97.         while True:
  99.             # Draw the state
  100.             self.print_state()
  102.             # Get a move from a player
  103.             if (self.player == 'X'): # AI
  105.                 # Print AI move
  106.                 print('Player X moving (AI) ...')
  108.                 # Get the best move
  109.                 max, py, px, depth = self.max(-sys.maxsize, sys.maxsize)
  111.                 # Get a heuristic move at cutoff
  112.                 print('Depth: {0}'.format(depth))
  113.                 if(depth > self.max_depth):
  114.                     py, px = self.get_best_move()
  116.                 # Make a move
  117.                 self.state[py][px] = 'X'
  119.                 # Check if the game has ended, break out from the loop in that case
  120.                 result = self.game_ended()
  121.                 if(result != None):
  122.                     break
  124.                 # Change turn
  125.                 self.player = 'O'
  127.             elif (self.player == 'O'): # Human player
  128.                 
  129.                 # Print turn
  130.                 print('Player O moving (Human) ...')
  132.                 # Get a recommended move
  133.                 min, py, px, depth = self.min(-sys.maxsize, sys.maxsize)
  135.                 # Get a heuristic move at cutoff
  136.                 print('Depth: {0}'.format(depth))
  137.                 if(depth > self.max_depth):
  138.                     py, px = self.get_best_move()
  140.                 # Print a recommendation
  141.                 print('Recommendation: {0}'.format(self.inverted_tiles.get((py, px))))
  143.                 # Get input
  144.                 number = int(input('Make a move (tile number): '))
  145.                 tile = self.tiles.get(number)
  147.                 # Check if the move is legal
  148.                 if(tile != None):
  149.                     
  150.                     # Make a move
  151.                     py, px = tile
  152.                     self.state[py][px] = 'O'
  154.                     # Check if the game has ended, break out from the loop in that case
  155.                     result = self.game_ended()
  156.                     if(result != None):
  157.                         break
  159.                     # Change turn
  160.                     self.player = 'X'
  162.                 else:
  163.                     print('Move is not legal, try again.')
  165.         # Print result
  166.         self.print_state()
  167.         print('Winner is player: {0}'.format(result))
  169.     # An evaluation function to get the best move based on heuristics
  170.     def get_best_move(self):
  172.         # Create an heuristic dictionary
  173.         heuristics = {}
  175.         # Get all empty cells
  176.         empty_cells = []
  177.         for y in range(self.rows):
  178.             for x in range(self.columns):
  179.                 if (self.state[y][x] == '.'):
  180.                     empty_cells.append((y, x))
  182.         # Loop empty positions
  183.         for empty in empty_cells:
  185.             # Get numbered position
  186.             number = self.inverted_tiles.get(empty)
  188.             # Loop winning positions
  189.             for win in self.winning_positions:
  191.                 # Check if number is in a winning position
  192.                 if(number in win):
  194.                     # Calculate the number of X:s and O:s in the winning position
  195.                     player_x = 0
  196.                     player_o = 0
  197.                     start_score = 1
  198.                     for box in win:
  200.                         # Get the position
  201.                         y, x = self.tiles[box]
  203.                         # Count X:s and O:s
  204.                         if(self.state[y][x] == 'X'):
  205.                             player_x += start_score if self.player == 'X' else start_score * 2
  206.                             start_score *= 10
  207.                         elif (self.state[y][x] == 'O'):
  208.                             player_o += start_score if self.player == 'O' else start_score * 2
  209.                             start_score *= 10
  211.                     # Save heuristic
  212.                     if(player_x == 0 or player_o == 0):
  214.                         # Calculate a score
  215.                         score = max(player_x, player_o) + start_score
  217.                         # Update the score
  218.                         if(heuristics.get(number) != None):
  219.                             heuristics[number] += score
  220.                         else:
  221.                             heuristics[number] = score
  223.         # Get the best move from the heuristic dictionary
  224.         best_move = random.choice(empty_cells)
  225.         best_count = -sys.maxsize
  226.         for key, value in heuristics.items():
  227.             if(value > best_count):
  228.                 best_move = self.tiles.get(key)
  229.                 best_count = value
  231.         # Return the best move
  232.         return best_move
  234.     # Check if the game has ended
  235.     def game_ended(self) -> str:
  237.         # Check if a player has won
  238.         result = self.player_has_won()
  239.         if(result != None):
  240.             return result
  242.         # Check if the board is full
  243.         for y in range(self.rows):
  244.             for x in range(self.columns):
  245.                 if (self.state[y][x] == '.'):
  246.                     return None
  248.         # Return a tie
  249.         return 'It is a tie!'
  250.        
  251.     # Check if a player has won
  252.     def player_has_won(self) -> str:
  253.         
  254.         # Loop the board
  255.         for y in range(self.rows):
  256.             for x in range(self.columns):
  257.                 
  258.                 # Loop vectors
  259.                 for vector in self.vectors:
  260.                     
  261.                     # Get the start position
  262.                     sy, sx = (y, x)
  264.                     # Get vector deltas
  265.                     dy, dx = vector
  267.                     # Create counters
  268.                     steps = 0
  269.                     player_x = 0
  270.                     player_o = 0
  272.                     # Loop until we are outside the board or have moved the number of steps in the goal
  273.                     while steps < self.goal:
  275.                         # Add steps
  276.                         steps += 1
  278.                         # Check if a player has a piece in the tile
  279.                         if(self.state[sy][sx] == 'X'):
  280.                             player_x += 1
  281.                         elif(self.state[sy][sx] == 'O'):
  282.                             player_o += 1
  284.                         # Update the position
  285.                         sy += dy
  286.                         sx += dx
  287.                 
  288.                         # Check if the loop should terminate
  289.                         if(sy < 0 or abs(sy) > self.max_row_index or sx < 0 or abs(sx) > self.max_columns_index):
  290.                             break
  292.                     # Check if we have a winner
  293.                     if(player_x >= self.goal):
  294.                         return 'X'
  295.                     elif(player_o >= self.goal):
  296.                         return 'O'
  298.         # Return None if no winner is found
  299.         return None
  301.     # Get a min value (O)
  302.     def min(self, alpha:int=-sys.maxsize, beta:int=sys.maxsize, depth:int=0):
  303.         
  304.         # Variables
  305.         min_value = sys.maxsize
  306.         by = None
  307.         bx = None
  308.         
  309.         # Check if the game has ended
  310.         result = self.game_ended()
  311.         if(result != None):
  312.             if result == 'X':
  313.                 return 1, 0, 0, depth
  314.             elif result == 'O':
  315.                 return -1, 0, 0, depth
  316.             elif result == 'It is a tie!':
  317.                 return 0, 0, 0, depth
  318.         elif(depth > self.max_depth):
  319.             return 0, 0, 0, depth
  321.         # Loop the board
  322.         for y in range(self.rows):
  323.             for x in range(self.columns):
  325.                 # Check if the tile is empty
  326.                 if (self.state[y][x] == '.'):
  328.                     # Make a move
  329.                     self.state[y][x] = 'O'
  331.                     # Get max value
  332.                     max, max_y, max_x, depth = self.max(alpha, beta, depth + 1)
  333.                     
  334.                     # Set min value to max value if it is lower than curren min value
  335.                     if (max < min_value):
  336.                         min_value = max
  337.                         by = y
  338.                         bx = x
  339.                         
  340.                     # Reset the tile
  341.                     self.state[y][x] = '.'
  343.                     # Do an alpha test
  344.                     if (min_value <= alpha):
  345.                         return min_value, bx, by, depth
  347.                     # Do a beta test
  348.                     if (min_value < beta):
  349.                         beta = min_value
  351.         # Return min value
  352.         return min_value, by, bx, depth
  354.     # Get max value (X)
  355.     def max(self, alpha:int=-sys.maxsize, beta:int=sys.maxsize, depth:int=0):
  357.         # Variables
  358.         max_value = -sys.maxsize
  359.         by = None
  360.         bx = None
  362.         # Check if the game has ended
  363.         result = self.game_ended()
  364.         if(result != None):
  365.             if result == 'X':
  366.                 return 1, 0, 0, depth
  367.             elif result == 'O':
  368.                 return -1, 0, 0, depth
  369.             elif result == 'It is a tie!':
  370.                 return 0, 0, 0, depth
  371.         elif(depth > self.max_depth):
  372.             return 0, 0, 0, depth
  374.         # Loop the board
  375.         for y in range(self.rows):
  376.             for x in range(self.columns):
  378.                 # Check if the current tile is empty
  379.                 if (self.state[y][x] == '.'):
  380.                     
  381.                     # Add a piece to the board
  382.                     self.state[y][x] = 'X'
  384.                     # Set max value to min value if min value is greater than current max value
  385.                     min, min_y, min_x, depth = self.min(alpha, beta, depth + 1)
  387.                     # Adjust the max value
  388.                     if (min > max_value):
  389.                         max_value = min
  390.                         by = y
  391.                         bx = x
  393.                     # Reset the tile
  394.                     self.state[y][x] = '.'
  396.                     # Do a beta test
  397.                     if (max_value >= beta):
  398.                         return max_value, bx, by, depth
  400.                     # Do an alpha test
  401.                     if (max_value > alpha):
  402.                         alpha = max_value
  404.         # Return max value
  405.         return max_value, by, bx, depth

  415.                     digit = str(self.inverted_tiles.get((y,x))) if len(str(self.inverted_tiles.get((y,x)))) > 1 else ' ' + str(self.inverted_tiles.get((y,x)))
  416.                     print('{0}  | '.format(digit), end='')
  417.             print()
  418.         print()
